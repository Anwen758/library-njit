随便用个文本软件打开[index.py] 把刚刚获得的[wechatSESS_ID] [SERVERID]，填入到对应位置[两个引号里边]

参考视频[部署腾讯云]，函数名称可以随便起

不清楚[SERVERID]的存活时间，第3步设置五分钟一次心跳可以保活不用修改
